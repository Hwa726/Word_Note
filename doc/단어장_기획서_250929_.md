## 1. 프로젝트 개요

### 1.1 프로젝트 배경
현대 학습 환경에서 어휘력은 학업 성취의 핵심 요소입니다. 특히 고등학교 수험생과 대학생들은 영어 시험, 자격증 준비, 전공 학습 등을 위해 방대한 양의 단어를 효율적으로 암기해야 합니다. 그러나 기존의 단어 학습 방법은 다음과 같은 한계를 가지고 있습니다:

- **일회성 암기의 한계**: 단순 반복 학습은 단기 기억에만 머물러 시험 후 빠르게 망각됨
- **개인차 무시**: 모든 학습자에게 동일한 학습량과 난이도를 제공하여 비효율적
- **학습 동기 저하**: 지루한 반복 학습으로 인한 학습 포기율 증가
- **학습 진도 파악 어려움**: 자신이 어떤 단어를 얼마나 알고 있는지 객관적 확인 불가

이러한 문제를 해결하기 위해 과학적 학습 이론에 기반한 개인화된 단어장 프로그램이 필요합니다.

### 1.2 프로젝트 필요성

#### 인지과학 기반 학습 효율성
- **간격 반복 학습(Spaced Repetition)**: 에빙하우스 망각곡선 이론에 따르면, 적절한 시간 간격으로 복습할 때 장기 기억으로 전환됩니다
- **능동적 회상(Active Recall)**: 수동적 읽기보다 능동적으로 기억을 떠올리는 과정이 학습 효과를 극대화합니다

#### 개인화 학습의 중요성
- 학습자마다 단어 숙지도, 학습 속도, 기억력이 다릅니다
- 개인의 학습 데이터를 분석하여 맞춤형 학습 계획을 제공함으로써 시간 대비 효율성을 극대화할 수 있습니다

#### 학업 성취도 향상
- 체계적인 어휘 학습은 독해력, 작문 능력, 의사소통 능력을 전반적으로 향상시킵니다
- 수능, TOEIC, TOEFL, 공무원 시험 등 각종 시험 성적 향상에 직접적인 영향을 미칩니다

#### 접근성 및 편의성
- 윈도우 PC 독립 프로그램으로 인터넷 연결 없이 언제든지 학습 가능
- 학생들이 가장 많은 시간을 보내는 PC 환경에서 자연스럽게 학습 습관 형성

## 2. 프로젝트 목표

### 2.1 최종 목표
**고등학교 수험생과 대학생이 일상적으로 사용하며 장기 기억을 촉진하고 학업 성적을 향상시킬 수 있는 개인화된 윈도우 PC 단어장 프로그램 개발**

### 2.2 세부 목표

#### 목표 1: 장기 기억 촉진
- 과학적으로 검증된 간격 반복 알고리즘 적용
- 개인의 망각 곡선을 분석하여 최적의 복습 타이밍 제공
- 단기 기억이 아닌 장기 기억으로의 전환율 향상

#### 목표 2: 학업 성적 향상
- 체계적인 단어 학습을 통한 독해력 및 어휘력 향상
- 시험 대비 효율적인 학습 전략 제공
- 실제 시험 성적 개선에 기여

#### 목표 3: 개인화된 학습 경험
- 사용자별 학습 데이터 수집 및 분석
- 개인의 학습 속도와 난이도에 맞춘 맞춤형 학습 제공
- 취약한 단어 집중 학습 기능

#### 목표 4: 일일 학습 습관 형성
- 매일 꾸준히 사용할 수 있는 학습 루틴 설계
- 학습 동기를 유지할 수 있는 피드백 시스템
- 간편하고 직관적인 사용자 경험 제공

## 3. 타겟 사용자

### 3.1 주요 타겟

#### 고등학교 수험생 (17-19세)
**학습 니즈:**
- 수능 영어 영역 대비 필수 어휘 암기
- 내신 시험 준비
- 영어 자격증 시험 (TOEIC, TOEFL, TEPS 등) 준비

**사용 환경:**
- 학교, 학원, 집에서 PC로 자습하는 시간 활용
- 짧은 시간에 집중적인 학습 선호
- 시험 일정에 맞춘 학습 계획 필요

**특성:**
- 명확한 학습 목표 (시험 점수)
- 시간 압박이 큰 학습 환경
- 효율성과 결과 중심 사고

#### 대학생 (19-26세)
**학습 니즈:**
- 전공 서적 독해를 위한 학술 어휘
- 토익, 토플 등 취업 대비 어학 시험 준비
- 원서 읽기, 논문 작성을 위한 고급 어휘

**사용 환경:**
- 강의실, 도서관, 기숙사에서 PC 사용
- 상대적으로 긴 학습 세션 가능
- 자기주도적 학습 스타일

**특성:**
- 장기적 관점의 학습 목표
- 깊이 있는 어휘 이해 추구
- 자기 관리 능력 보유

### 3.2 사용자 페르소나

- 목표: 수능 영어 1등급

**페르소나 2: 대학생 이서연 (22세, 대학 3학년)**
- 목표: 토익 900점 이상, 교환학생 준비
- 학습 가능 시간: 하루 1시간 내외
- 약 5,000개 단어 암기 필요
- 단어의 뜻뿐만 아니라 활용법도 익히고 싶음
- 장기적으로 꾸준히 학습할 수 있는 시스템 원함

## 4. 핵심 가치 제안

### 4.1 차별화 요소

#### 과학적 학습 알고리즘
- 단순 암기가 아닌 인지과학 기반의 학습 방법론 적용
- 개인별 망각 곡선 데이터를 활용한 최적화된 복습 스케줄

#### 완전한 개인화
- 사용자의 학습 이력을 실시간으로 분석
- 잘 아는 단어는 복습 주기를 늘리고, 어려운 단어는 집중 학습
- "나만의 취약점"을 자동으로 파악하고 보완

#### PC 환경 최적화
- 윈도우 PC에서 독립 실행 가능한 프로그램
- 인터넷 연결 불필요, 언제 어디서나 오프라인 학습 가능
- 학생들의 주요 학습 공간인 PC 환경에 최적화

#### 간편하고 직관적인 UX
- 복잡한 설정 없이 바로 시작 가능
- 학습에만 집중할 수 있는 미니멀한 인터페이스
- 빠른 학습 세션 (5~30분) 지원

### 4.2 기대 효과

#### 학습자 관점
- **시간 효율성 향상**: 개인 맞춤 학습으로 불필요한 반복 제거
- **장기 기억력 강화**: 과학적 복습 알고리즘으로 망각 방지
- **학습 동기 유지**: 명확한 진도 확인과 성취감 제공
- **시험 성적 향상**: 체계적 어휘 학습을 통한 실질적 성적 개선

#### 교육적 관점
- 효과적인 어휘 습득 방법론 제시
- 자기주도 학습 능력 배양
- 데이터 기반 학습 전략 수립 가능
