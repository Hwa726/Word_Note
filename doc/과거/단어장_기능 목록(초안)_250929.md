## 단어장 기능 목록
주요 기능
### 단어 암기(플래시카드)
   1. 영->한 / 한->영
   2. 순차 / 랜덤 / 개인화(많이 틀리는 문제의 비중 상승)
   3. 답을 함께 보여주거나 블라인드 처리하거나
   4. 잘 틀리는 단어 색깔 표시
### 단어 관리
   1. 단어 목록 조회, 
   2. 단어 업로드/다운로드(csv로)
   3. 단어 변경, 추가
   4. (수정기능)메모기능, 수정한 날짜, 수정된 단어 목록 표시
   5. 시험 후 오답 노트 자동 생성 및 복습 모드 연결
   6. 답변 제한시간 설정
### 단어 시험
   1. 랜덤 순서 혹은 오답률 등을 기반한 개인화된 순서
   2. 단답형
   3. 객관식
### 학습관리
   1. (하루)학습 목표 지정
   2. 학습 통계, 오늘 통계, 전체 통계
### 공통 기능
   1. 단어 즐겨찾기(단어 암기, 관리, 시험 중에)
   2. 개인화 : 자주 틀리는 단어를 기록하고, 암기할 때나 시험 볼 때 좀 더 나오도록 함.
### 향후
   1. 커뮤니티(비교, 공유, 대결)
   2. 학습집중 모드(타 프로그램 알람 제어)
   3. 개인 수준측정
   4. 모바일 연동
   5. 다국어 추가(여러 나라 단어장들의 구분 관리)
   6. 발음 읽어주기(TTS)
   7. 로그인
   8. 단어 카테고리 관리
   9. 목표 설정에 유연성 추가(예: 주간 목표, 특정 시험 D-Day 연동).
   10. 통계 시각화(예: 그래프, 차트)로 직관성 강화
   11. 테마 설정: 다크/라이트 모드 지원으로 사용자 편의성 증대
   12. 모의고사 모드
