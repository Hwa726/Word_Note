## 1. 화면 설계 개요

### 1.1 설계 목적
사용자가 직관적으로 사용할 수 있는 UI/UX 설계 및 화면별 기능 정의

### 1.2 화면 구성 원칙
- **직관성**: 3클릭 이내 모든 기능 접근
- **일관성**: 동일한 UI 패턴 반복 사용
- **피드백**: 모든 액션에 즉시 반응
- **최소화**: 학습에 집중할 수 있도록 불필요한 요소 제거

### 1.3 주요 설계 결정 사항 (v1.2)
✅ **메인 윈도우**: 탭 기반 네비게이션 (QTabWidget 사용)  
✅ **플래시카드**: 플립 카드 애니메이션 방식  
✅ **단어 목록**: 메모 + 최종학습일 + 오답률 컬럼 포함  
✅ **시험**: 설정 → 진행 → 결과의 3단계 화면 전환

### 1.4 전체 화면 구조

```
메인 윈도우 (QTabWidget)
├── [대시보드] 탭
├── [단어관리] 탭
│   ├── 단어 목록 (메인)
│   ├── 단어 추가 다이얼로그
│   ├── 단어 수정 다이얼로그
│   └── CSV 임포트/엑스포트 다이얼로그
├── [학습] 탭
│   ├── 학습 설정 화면
│   └── 플래시카드 화면 (플립 카드 방식)
├── [시험] 탭
│   ├── 시험 설정 화면
│   ├── 시험 진행 화면 (단답형/객관식)
│   └── 시험 결과 화면
├── [오답노트] 탭
└── [통계] 탭
```

## 2. 메인 윈도우 (탭 기반)

### 2.1 레이아웃 구조

```
┌───────────────────────────────────────────────────────────┐
│  스마트 단어장                                    [─][□][X] │
├───────────────────────────────────────────────────────────┤
│  메뉴: 파일 | 편집 | 보기 | 학습 | 설정 | 도움말            │
├───────────────────────────────────────────────────────────┤
│  툴바: [+단어추가] [플래시카드] [시험] [CSV가져오기] [백업]  │
├───────────────────────────────────────────────────────────┤
│                                                           │
│  📊 대시보드 | 📚 단어관리 | 🎴 학습 | 📝 시험 | ❌ 오답노트 | 📈 통계 |
│  ═══════════════════════════════════════════════════       │
│                                                           │
│                    [탭 컨텐츠 영역]                         │
│                  (선택된 탭의 내용 표시)                    │
│                                                           │
├───────────────────────────────────────────────────────────┤
│  총 1,234개 | 오늘 학습: 50개 | 정답률: 85% | 14:45      │
└───────────────────────────────────────────────────────────┘
```

### 2.2 탭 구성

| 탭 | 아이콘 | 탭 이름 | 단축키 | 기능 |
|---|--------|---------|--------|------|
| 1 | 📊 | 대시보드 | Ctrl+1 | 학습 현황 한눈에 보기 |
| 2 | 📚 | 단어관리 | Ctrl+2 | 단어 CRUD, CSV 관리 |
| 3 | 🎴 | 학습 | Ctrl+3 | 플래시카드 학습 |
| 4 | 📝 | 시험 | Ctrl+4 | 단답형/객관식 시험 |
| 5 | ❌ | 오답노트 | Ctrl+5 | 틀린 단어 관리 |
| 6 | 📈 | 통계 | Ctrl+6 | 학습 통계 및 분석 |

### 2.3 메뉴바 구성

**파일 메뉴**
- CSV 가져오기 (Ctrl+O)
- CSV 내보내기 (Ctrl+E)
- 백업하기 (Ctrl+B)
- 복원하기
- 종료 (Ctrl+Q)

**편집 메뉴**
- 단어 추가 (Ctrl+N)
- 단어 수정 (Enter)
- 단어 삭제 (Delete)
- 전체 선택 (Ctrl+A)

**보기 메뉴**
- 대시보드 (Ctrl+1)
- 단어관리 (Ctrl+2)
- 학습 (Ctrl+3)
- 시험 (Ctrl+4)
- 오답노트 (Ctrl+5)
- 통계 (Ctrl+6)

**학습 메뉴**
- 플래시카드 시작
- 시험 시작
- 오답노트 복습

**설정 메뉴**
- 학습 목표 설정
- UI 설정
- 백업 설정

**도움말 메뉴**
- 사용 가이드 (F1)
- 단축키 목록
- 프로그램 정보

### 2.4 상태바

```
[좌측] 총 1,234개 | 오늘 학습: 50개 | 정답률: 85%
[우측] 14:45
```

## 8. 인터랙션 상세

### 8.1 플래시카드 인터랙션

**카드 뒤집기**:
1. 앞면(문제) 상태에서 카드 클릭 또는 Space 키
2. 3D 플립 애니메이션 (0.3초)
3. 뒷면(정답) 표시

**정답/오답 판정**:
1. 뒷면에서 [맞음] 또는 [틀림] 버튼 클릭
2. 학습 이력 데이터베이스 기록
3. 통계 업데이트
4. 2초 대기 후 자동으로 다음 카드

**답변 제한 시간**:
- 설정된 시간 초과 시 자동으로 오답 처리
- 남은 시간 카운트다운 표시

### 8.2 단어 목록 인터랙션

**단어 선택**:
- 단일 클릭: 선택
- 더블 클릭: 수정 다이얼로그
- 우클릭: 컨텍스트 메뉴 (수정/삭제/즐겨찾기)

**즐겨찾기**:
- ⭐ 아이콘 클릭: 즐겨찾기 토글
- 즐겨찾기 상태 즉시 DB 반영

**검색**:
- 입력 중 300ms 디바운스
- 영어/한국어/메모에서 검색
- 부분 일치 검색

### 8.3 시험 인터랙션

**문제 이동**:
- [이전]/[다음] 버튼
- 화살표 키 (←/→)
- 문제 번호 클릭

**답변 입력**:
- 단답형: 텍스트 입력
- 객관식: 라디오 버튼 선택
- 건너뛰기: 답변 없이 다음 문제

**시험 제출**:
1. [시험 제출] 버튼 클릭
2. 확인 다이얼로그 표시
3. [제출] 확인 시 채점
4. 결과 화면으로 전환

### 9.1 에러 메시지 다이얼로그

- 길이 초과: "최대 글자 수를 초과했습니다"
- CSV 오류: "CSV 파일 형식이 잘못되었습니다"

### 9.2 성공 메시지 (토스트)

```
┌────────────────────────┐
```

- "N개 단어를 삭제했습니다"

- "백업이 완료되었습니다"

- 오답 노트 전체 삭제

## 10. 접근성 고려사항

### 10.1 키보드 네비게이션
- **Tab 키**: 다음 컨트롤로 포커스 이동
- **Shift+Tab**: 이전 컨트롤로 포커스 이동
- **Enter**: 버튼 활성화, 입력 완료
- **Esc**: 다이얼로그 닫기, 작업 취소
- 포커스 표시: 파란 테두리

### 10.2 시각적 피드백
- 모든 클릭 가능 요소: 호버 시 커서 변경 (pointer)
- 모든 버튼: 비활성 상태 명확히 표시 (회색, 투명도)
- 모든 입력 필드: 포커스 시 테두리 색상 변경, 오류 시 빨간 테두리

### 10.3 스크린 리더 지원
- 모든 중요 요소에 alt 텍스트
- 진행 상황 알림: "5 / 50 문제 진행 중"
